=======================================
Instrucciones de Ejecución - Audiobook Studio
=======================================

Esta es una aplicación portable para Windows, lo que significa que no requiere instalación.

Para ejecutar el programa, sigue estos sencillos pasos:

1.  Asegúrate de que todos los archivos y carpetas (especialmente la carpeta "_internal") se encuentren en el mismo directorio que el archivo ejecutable.

2.  Busca el archivo llamado "AudiobookStudio.exe".

3.  Haz doble clic sobre "AudiobookStudio.exe" para iniciar la aplicación.

¡Y eso es todo! El programa debería abrirse y estar listo para usarse.

---
Nota: Si la aplicación no se inicia, es posible que tu antivirus o Windows Defender la esté bloqueando. Si aparece una advertencia de seguridad, deberás concederle los permisos necesarios para ejecutarse.
